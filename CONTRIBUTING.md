# Contributing to Tour & Travel Project

First off, thank you for considering contributing to this project! It's people like you that make open-source such a great community. 

## How to Contribute

### 1. Fork and Clone
1. Fork the repository to your own GitHub account.
2. Clone the project to your machine:
   ```bash
   git clone https://github.com/YOUR-USERNAME/Tour-Project.git
   ```

### 2. Create a Branch
Always create a new branch for your work. Use a descriptive name:
```bash
git checkout -b feature/interactive-map
# or
git checkout -b bugfix/navbar-visibility
```

### 3. Make Your Changes
- Ensure your HTML is semantic and properly indented.
- Keep CSS organized. If you add new styles, consider placing them in the appropriate section of `style.css`.
- Make sure the website remains fully responsive on mobile devices!

### 4. Commit Your Changes
Write clear and concise commit messages:
```bash
git commit -m "Add responsive user testimonials carousel"
```

### 5. Open a Pull Request
- Push your branch to your forked repository.
- Open a Pull Request against our `main` branch.
- **IMPORTANT**: Make sure to fill out the Pull Request Template provided. Your PR will be reviewed based on those checklist items.

## Code Review Guidelines
When your PR is opened, maintainers will review it. We look for:
- **Functionality**: Does the code do what it's supposed to do?
- **Responsiveness**: Does it look good on mobile and desktop?
- **Readability**: Is the code clean and easy to understand?
- **No breaking changes**: Does it break existing UI elements?

Thank you for contributing!
